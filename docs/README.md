# Documentación

La carpeta `docs/` contiene la documentación de diseño y de desarrollo del proyecto FrontEnd-event-platform.

Archivos incluidos:

- `SETUP.md` — Cómo ejecutar el frontend localmente y notas sobre el backend (implementado en un repositorio C# separado).
- `MOCKS.md` — Descripción de los adaptadores mock existentes y cómo reemplazarlos por adaptadores reales que llamen a la API.
- `ARCHITECTURE.md` — Visión general de la arquitectura del proyecto y responsabilidades de las carpetas.
- `API.md` — Enlace rápido al archivo `API_ENDPOINTS.md` en la raíz con las rutas y contratos de los endpoints.

Consulta los archivos individuales para más detalles.
